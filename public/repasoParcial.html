Ejercicio 3 

program ejercicio3;
var 
  cant, sum : integer;
begin
  cant := 1; // 1, 2, 3, 4
  sum := 0;
  while (cant < 4) or (sum < 10) do
  begin
    writeln('cant: ', cant, ' sum: ', sum);
    cant := cant + 1;
    sum := sum + 1;
  end;
  writeln('sum: ', sum);
end.

La opcion correcta es 10

OR -> El ciclo se ejecuta mientras al menos una de las condiciones sea verdadera.
En este caso, el ciclo se ejecutará mientras cant < 4 o sum < 10.

0 
1 2 3
El resultado final es: 0

Con dos condiciones en el while
and -> Las dos condiciones tienen que ser verdaderas para que el ciclo se ejecute
- Si una de las condiciones es falsa, el ciclo no se ejecuta
or -> Al menos una de las condiciones tiene que ser verdadera para que el ciclo se ejecute
- Si ambas condiciones son falsas, el ciclo no se ejecuta
